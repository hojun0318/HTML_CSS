# HTML (Hyper Text Markup Language)

## Hyper Text
- 참조(하이퍼링크)를 통해 사용자가 한 문서에서 다른 문서로 즉시 접근할 수 있는 텍스트

## Markup Language
- 태그 등을 이용하여 문서나 데이터의 구조를 명시하는 언어
    - 대표적인 예 - HTML, Markdown
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Hello, HTML</title>
</head>
<body>
</body>
</html>
```
